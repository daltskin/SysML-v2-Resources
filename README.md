# SysML v2 Resources

A curated list of useful resources for learning and working with [SysML v2](https://www.omg.org/spec/SysML/2.0), the next-generation systems modeling language from OMG.

> **Disclaimer:** Inclusion in this list does not constitute an endorsement of any product, service, or organization. These resources are provided for informational purposes only.

## Documentation

- [OMG SysML v2 Specification](https://www.omg.org/spec/SysML/2.0) — Official specification (Language & Transformation PDFs)
- [OMG KerML Specification](https://www.omg.org/spec/KerML) — Kernel Modeling Language underpinning SysML v2
- [SysML v2 Release Repository](https://github.com/Systems-Modeling/SysML-v2-Release) — Latest incremental release including the standard library, grammar, and example models
- [SysML v2 Pilot Implementation](https://github.com/Systems-Modeling/SysML-v2-Pilot-Implementation/) - Proof-of-concept pilot implementation of the SysML v2 textual notation and visualization


## Learning Resources

- [Advent of SysML v2](https://sensmetry.com/advent-of-sysml-v2/) — Free 25-lesson self-paced course with videos, blog posts, and hands-on exercises covering language fundamentals to real-world workflows
- [MBSE4U — SysML v2](https://mbse4u.com/sysml-v2/) — Blog posts, tool overviews, and SysML v2 methodology articles
- [MBSE Podcast](https://mbse-podcast.rocks/) — Podcast covering MBSE topics including SysML v2
- [OMG Systems Modeling Community](https://www.omg.org/communities/systems-modeling-community.htm) — Official OMG community hub for SysML

## Authoring Tools

- [SysML v2.0 Language Support](https://marketplace.visualstudio.com/items?itemName=JamieD.sysml-v2-support) — Free VS Code extension with full LSP, interactive diagrams, model explorer, feature inspector, dashboard, MCP server and more by JamieD
- [Syside Editor: SysML v2 Essential](https://marketplace.visualstudio.com/items?itemName=sensmetry.syside-editor) — Free VS Code extension for SysML v2 textual editing with validation, completion, navigation, and formatting by Sensmetry
- [Syside Modeler: SysML v2 Professional](https://syside.sensmetry.com/) — Premium SysML v2 modeling environment with diagrams and CLI (Sensmetry) by Sensmetry
- [Syside Automator](https://docs.sensmetry.com/automator/index.html) — Python API for SysML v2 model processing, automation, and code generation
- [System Modelling Language v2](https://marketplace.visualstudio.com/items?itemName=Ellidiss.sysml-ellidiss) - SysML v2 support for Visual Studio Code by Ellidiss
- [SysML v2 Pilot Implementation (Eclipse)](https://github.com/Systems-Modeling/SysML-v2-Pilot-Implementation) — Eclipse-based editor with Xtext grammar and PlantUML visualization
- [SysML v2 Jupyter Kernel](https://github.com/Systems-Modeling/SysML-v2-Pilot-Implementation/tree/master/org.omg.sysml.jupyter.kernel) — Interactive SysML v2 notebook kernel for Jupyter
- [Sysand](https://sysand.org/) — Open-source SysML v2 package manager and package index

## Modelling Tools

- [OMG SysML v2 Tools List](https://www.omg.org/sysml/sysmlv2/sysml-tool/) — Official OMG directory of SysML v2 tool vendors
- [MBSE4u Tools list](https://mbse4u.com/mbse-tools/) - MBSE4u tool vendors
- [Eclipse SysON](https://mbse-syson.org/) — Open-source, web-based graphical SysML v2 modelling tool (Obeo / Eclipse Foundation), with Capella interoperability
- [SysGit](https://www.sysgit.io/) — Git-based collaborative platform for SysML v2 modelling, requirements management, and systems engineering automation
- [CATIA Magic / Cameo (Dassault Systèmes)](https://www.3ds.com/products/catia/no-magic) — Enterprise MBSE tool with SysML v2 support
- [Enterprise Architect / Trechoro (Sparx Systems)](https://sparxsystems.com/) — Modelling platform with SysML v2 support
- [Rhapsody (IBM)](https://www.ibm.com/products/systems-design-rhapsody) — Systems engineering tool with SysML v2 support
- [Systems Modeler with SysML v2 (Siemens)](https://plm.sw.siemens.com/) — Web-based collaborative SysML v2 modelling platform integrated with Teamcenter
- [PTC Modeler (PTC)](https://www.ptc.com/) — MBSE solution supporting SysML v1/v2, UML, and UAF with graphical and textual notations
- [System Architecture Modeler (Ansys/Synopsys)](https://www.ansys.com/) — Cloud-native SysML v2 tool with graphical editing, real-time collaboration, and simulation
- [Dalus](https://www.dalus.ai/) — AI-native MBSE platform built on SysML v2 for architecture, requirements, and hazard analysis
- [Mg (Mgnite)](https://www.mgnite.com/) — SysML v2 modelling integrating Excel, PowerPoint, MATLAB/Simulink, and DOORS Next in JupyterLab
- [SystemsModeling.com (Model Driven Solutions)](https://www.systemsmodeling.com/) — Web-based platform for authoring, validating, and visualising SysML v2 models

## Language Server Protocols

- [SysML v2 Pilot Implementation](https://github.com/Systems-Modeling/SysML-v2-Pilot-Implementation) — Official Xtext-based parser and LSP server (Eclipse/Jupyter)
- [sysml-v2-lsp](https://github.com/daltskin/sysml-v2-lsp) — ANTLR4-based LSP server for SysML v2 with diagnostics, completions, semantic tokens, and MCP server for AI integration, complete with sample python & web clients
- [Syster](https://github.com/jade-codes/syster) — Rust-based SysML v2 parser and LSP with semantic analysis, Python bindings, and VS Code extensions
- [SysML v2 ANTLR4 Grammar](https://github.com/antlr/grammars-v4/tree/master/sysml-v2) — ANTLR4 lexer and parser grammars for SysML v2 textual notation

## API
- [SysML v2 API Cookbook](https://github.com/Systems-Modeling/SysML-v2-API-Cookbook) — Jupyter Notebook recipes for using the SysML v2 API
- [SysML v2 API Services](https://github.com/Systems-Modeling/SysML-v2-API-Services) — Pilot implementation of the Systems Modeling API and Services
- [SysML v2 API Java Client](https://github.com/Systems-Modeling/SysML-v2-API-Java-Client) — Java client for the Systems Modeling API
- [SysML v2 API Python Client](https://github.com/Systems-Modeling/SysML-v2-API-Python-Client) — Python client for the Systems Modeling API
